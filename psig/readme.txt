私钥格式:
{
    UUID:               uuid,
    En(私钥 || 口令):   key,
    公钥:               keypub,
    签名:               headhersig
}

签名格式:
{
    文件哈希值:         hash,
    签名时间:           time,
    原UUID:             ouuid,
    实际签名人UUID:     suuid,
    签名:               sig
}

Client表格式: 继承 $私钥格式
{
    uuid: $私钥格式,
    ...
}

Agent表格式:
{
    uuid:{
        UUID:               uuid,
        公钥:               keypub
    },
    ...
}

User表格式:
{
    UUID:               uuid, //主键
    创建时间:           btime,
    注销时间:           dtime, //默认0
    公钥:               keypub
}

Deal表格式:
{
    代理人:             suuid, //实际执行签名操作的人
    被代理人:           ouuid, //(suuid,ouuid)构成复合主键
    创建时间:           btime,
    注销时间:           dtime, //默认0
    公钥:               keypub
}

错误代码约定:
{
    0     : 运行正确,
    0x4000: 未定义错误类型,
    
    0x4001: 口令错误,
    
    0x4011: 文件读写失败,

    
    0x4F01: 文件哈希值不符合,
    0x4F02: 代理权限过期或不存在,
    0x4F03: UUID不存在,
    0x4F04: 签名不符        
}
